# git 技巧整理



## 删除一个已推送到远端的tag 并重建tag

```shell
# 删除本地分支
git tag -d v1.0.0
# 更新远端
git push origin :refs/tags/v1.0.0
# 查看最新一次提交的 commit 和 消息
git log -1 --format="Commit: %H%n%B"
# 创建新tag
git tag -a v1.0.0 commit -m "tag message"
# 推送提交
git push origin 
# 推送tag
git push origin v1.0.0

```
