# 搭建自己的博客教程

## hugo

### 安装

```shell
# 安装go可用
go clean -modcache && CGO_ENABLED=1 go install -tags extended github.com/gohugoio/hugo@latest
```

### 链接

* [hugo-paper主题](https://github.com/nanxiaobei/hugo-paper)
* [hexo教程](https://blog.csdn.net/yaorongke/article/details/119089190)
* [hugo教程](https://hugo.opendocs.io/)

## zola

### 安装

```shell
# 安装rust可用
cargo install --locked --git https://github.com/getzola/zola
```

### 链接

* [zola官网](https://www.getzola.org/documentation/getting-started/overview/)
